-  **Ace of Diamonds** - pixel (Pixel Phone - Mobile)
-  **Two of Diamonds** - two-of-diamonds (Physical Server - Proxmox VE Host)
-  **Three of Diamonds** - three-of-diamonds (AI Server - AI/ML Workloads)
-  **Jack of Diamonds** - jack-of-diamonds (MeLE Mini PC - Primary Services Host / DNS)

-  **King of Diamonds** - king-of-diamonds (Proxmox VM - TrueNAS Storage)
